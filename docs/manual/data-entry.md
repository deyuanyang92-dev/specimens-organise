# 数据录入

## 录入任务门控

左侧入库编号面板顶部的 **开始录入任务** 是硬门控 —— 不点开始，**＋新增入库编号** 按钮 disabled。

![录入任务门控](TODO_screenshots/data-entry-task-gate.png)

任务字段：录入人 / 用途 / 备注。开始后会写一条「任务开始」到编号分发记录；结束写「任务结束」。

## 沿用上条信息

新增入库编号时，以下字段会自动沿用前一条：

- 标本存放位置
- 信息录入人员
- 核对人员
- 保存方式
- 备注

这是**固定规则**，没有开关（旧版工具栏曾有「沿用上条信息」勾选项，规范化后移除）。

## 自动派生

填 `管内编号*` 时，以下字段自动派生：

- **采集日期** ← 管内编号尾部的 `YYMMDD`
- **采集地点缩写*** ← 管内编号中段的地点码
- **保存方式** ← 管内编号中的字母代码（R = 70% 乙醇、E = EtOH 等）

如管内编号 `QD-LSD-SC001-1-R-250923` → 日期 `2025-09-23`、地点 `LSD`、保存方式 `R`。

## 字段帮助 (?)

每个有说明的字段旁有一个低调的灰色 **?** 小按钮。hover 看摘要，点击弹完整说明：

![字段帮助按钮](TODO_screenshots/data-entry-field-help.png)

说明来源是软件自带的 `字段模版/数据录入字段及字段说明.xlsx` Sheet2（大类 / 字段名称 / 填写示例 / 字段说明 / 其他要求）。

应用内每个字段旁都有 ? 按钮可查；完整列表见 **菜单栏 → 帮助 → 字段填写说明速查**。

## 自动保存

工具栏右段 **自动保存** 勾选项控制是否在输入后自动写入（默认开）。取消勾选时仍可点字段下方「保存」手动写入。

## 批量设置标本信息

左侧入库编号列表多选 → 右键 **批量设置标本信息** 一次性写入多条凭证的同一字段（每条凭证一条 undo 记录）。
