# 开始使用

## 首次启动

![首次启动主界面](TODO_screenshots/getting-started-first-launch.png)

启动后，程序会自动定位工作区（last-used → 当前目录 → 可执行文件父目录）。如果未找到，菜单栏 **工具 → 切换工作区** 选择或新建一个。

工作区必须是普通文件夹，**不能**是：
- 磁盘根目录（如 `C:\`、`/`）
- 用户主目录
- 软件自带的 `build/` / `dist/` / `releases/` 子目录

## 基本概念

- **工作区**：一个文件夹，存放某个项目 / 某次野外采集的全部标本数据，便携可拷贝。
- **入库编号**（凭证号）：每条标本的唯一 ID，形如 `YZZ000001`、`BMNH000001`。
- **管内编号**：来自实际试管标签，如 `QD-LSD-SC001-1-R-250923`，可派生采集日期与地点缩写。
- **照片**：与入库编号关联的图像文件，被复制（去重）到工作区 `照片/` 目录。

## 第一次录入

1. 工具栏点 **切换工作区** 选目录。
2. 左侧面板顶部点 **开始录入任务**，填录入人 / 用途。
3. 点 **＋新增入库编号** 生成下一个凭证。
4. 在右侧填管内编号 → 采集日期 / 地点 / 保存方式自动派生。
5. 拖入照片或 **导入数据** → 自动关联。
6. 完成后点 **结束录入任务**。
