# 照片管理

## 关联照片

三种方式把照片加入凭证：

1. **拖入**：把图片文件拖到主窗口或照片表。
2. **导入数据**：工具栏 **导入数据** 自动从指定目录扫照片并按文件名匹配凭证。
3. **手动添加**：照片表右键 → 添加。

![照片关联](TODO_screenshots/photos-attach.png)

非工作区内的照片会被**复制**到 `照片/` 目录（按 SHA256 去重；同名不同内容加 `_2` `_3` 后缀）。工作区内照片保留原位置，仅记录相对路径。

## 预览 / 网格

主窗口中央是照片预览：

- **网格模式**：2 / 4 / 6 / 8 网格，多张同时看
- **单张模式**：双击网格放大；F 键适配；100% 显示

格内 Ctrl/Shift 多选支持。Esc 返回网格。

## 取消关联（删除）

> **重要**：UI 里的 "**删除**" 一律标 "**取消关联**"。

「取消关联」只删 `照片信息.xlsx` 的记录行，必要时清理工作区 `照片/` 里的存档副本（如果无其它凭证引用）。**绝不**删除用户的原始导入文件。

入口：
- 照片表右键 → 取消关联
- 网格右键 → 取消关联
- 单张预览右键 → 取消关联

## 照片检索

工具栏 → **图片搜索**：按管内编号核心标识（如 `LSD-SC001-000001`）匹配文件名。默认搜 TIFF，可切 JPG / 全部。

## 照片文件名填充

`Ctrl+Q`（默认，可在 **设置** 里改）— 从当前照片文件名解析管内编号 / 采集日期等填到当前凭证。
