# 入库汇总

工具栏 / 工具菜单 → **入库汇总**。

非模态、单实例、只读宽表，展示每个凭证的全部信息（标本 + 分类 + 照片数 + 照片聚合列）。

![入库汇总](TODO_screenshots/ingest-summary.png)

## 列

- 入库编号*（只读）
- 标本信息全部字段
- 分类信息全部字段（注意「分类备注」列名以避免与「备注」歧义）
- 照片数（只读）
- 照片文件名 / 照片绝对路径 / 照片描述（默认隐藏 — 同凭证多照片用 `; ` 拼接）

## 操作

- **左键表头**：排序
- **右键任意处**（表头或表体）：开筛选菜单
- **双击行**：跳到主窗口对应凭证（清掉主窗口的搜索/筛选，翻到对应页，选中并载入编辑器）
- **Ctrl/Shift 多选**：Windows 风格
- **右键表体** → **导出选中照片**：把选中凭证的全部照片走批量导出

## 搜索 / 筛选 / 导入编号列表

顶部搜索框全局过滤。各列筛选菜单可单独筛。 **导入编号列表** 按钮 — 粘贴或从 txt/csv/xlsx 加载凭证号列表，作为额外的 voucher 过滤。

## 列设置

导航栏 **列设置** 按钮开合右侧列可见性面板（默认隐藏）。可见列状态保存到 `settings.json` 的 `summary_visible_columns`。

## 导出

刷新 / Excel / CSV 导出按钮 — 导出**当前筛选 + 可见列**的快照。
