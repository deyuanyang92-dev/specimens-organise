# 工作区与数据

工作区是一个普通文件夹，结构高度规范化，可整体拷贝 / 备份。

## 文件结构

```
某工作区/
├── 数据/
│   ├── 标本信息.xlsx
│   ├── 照片信息.xlsx
│   ├── 分类信息.xlsx
│   ├── 编号索引.xlsx        # UUID + 指纹
│   ├── 修改记录.xlsx
│   ├── 操作记录.xlsx        # 撤销 / 重做日志
│   ├── 数据版本记录.xlsx
│   ├── 编号分发记录.xlsx     # 编号分发审计
│   ├── 工作区配置.json
│   ├── 数据版本/             # 数据快照
│   ├── 缩略图缓存/
│   └── .workspace.lock      # 进程级文件锁
├── 照片/                     # 内容哈希去重存档
└── （可选）字段模版/          # 兼容老工作区，运行时不再使用
```

## 关键点

- **Excel 全量覆写**：每次保存 openpyxl 重新创建 Workbook。**不要**在程序运行时手动改 Excel —— 外部改动会被下一次保存覆盖。
- **照片去重**：同 SHA256 的照片只存一份；同名不同内容的照片会自动加数字后缀（`same.jpg` → `same_2.jpg`）。
- **文件锁**：工作区一次只能被一个进程打开。陈旧锁（PID 已死 或 >10 分钟）会自动释放。

## 切换工作区

工具栏 → **切换工作区** 或 **导入工作区**。

## 导入 / 导出整个工作区

- **导入工作区**：把另一个工作区合并进当前工作区（基于指纹去重）。
- **导出数据**：把整个 `数据/` 与 `照片/` 拷出去 — 实际就是文件拷贝。

合并场景细节见 [合并 / 导入示例](import-merge.md)。
