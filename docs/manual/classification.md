# 分类信息与 WoRMS

## 分类信息面板

右侧 **分类信息** 面板字段：

- 物种中文名 / 物种拉丁
- 属名（拉丁名首词派生）
- 科* / 科拉丁
- 目 / 纲 / 门
- 分类备注（可附 WoRMS:AphiaID）

## 物种 / 科 自动匹配

填 `物种中文名` 或 `物种拉丁` 触发预设搜索：在软件自带 `字段模版/表格信息预设字段.xlsx` 里按
精确 > 前缀 > 包含的排序查 SpeciesMatch（含中文 / 拉丁 / 科 / 科拉丁），命中后自动填入其它分类字段。

只填 `科*` 时退化为 FamilyMatch。

> 预设文件随软件分发；任何工作区都能用，无需自行复制 `字段模版/`。

## WoRMS 物种分类

工具菜单 → **WoRMS 分类匹配** / **WoRMS 分类浏览** / **WoRMS 本地数据库管理**，详见 [WoRMS 物种分类](worms.md)。

匹配成功后可把 `WoRMS:AphiaID` 追加到「分类备注」，供日后 [Darwin Core 导出](dwc-export.md) 用作 `taxonID`。

## 预设缺失警告

软件启动时如检测到 `字段模版/表格信息预设字段.xlsx` 缺失，主窗口工具栏下方会显示**黄色警告 banner**（永久可见，提示自动匹配不可用）。这通常是源码运行但没拉完整仓库的情况；重装 / 重新拉取仓库后 banner 消失。
