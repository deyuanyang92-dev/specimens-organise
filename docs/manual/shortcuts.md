# 快捷键速查

> 应用内入口：**菜单栏 → 帮助 → 快捷键速查**（弹窗内容来自此文档与 ui.py 的 QShortcut 绑定）

| 快捷键 | 功能 |
|---|---|
| `Ctrl + A` | 全选凭证（入库编号列表内） |
| `Ctrl + Z` | 撤回 |
| `Ctrl + Y` / `Ctrl + Shift + Z` | 重做 |
| `Ctrl + =`, `Ctrl + +` | 界面字体放大 |
| `Ctrl + -` | 界面字体缩小 |
| `Ctrl + 0` | 界面字体复位 |
| `F` | 照片适配窗口 |
| `Esc` | 返回照片网格 |
| `F2` | 编辑照片表当前单元格 |
| `Ctrl + C` | 复制照片表选中（在照片表内） |
| 可自定义（默认 `Ctrl + Q`） | 从照片文件名填充标本信息（设置内改） |

> 右键菜单内的操作（导出选中、批量设置标本信息等）通常没有全局快捷键，请通过菜单或右键访问。
